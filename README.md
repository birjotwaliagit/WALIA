# WALIA

This repository contains a small web utility to generate Elementor-compatible JSON files from plain text case studies. Upload your base template, paste the raw text, and click **Generate JSON** to preview and download the filled template.

### Features

- Upload any Elementor template JSON
- Paste raw case study text and parse it automatically
- Load an example case study for quick demo
- Toggle light or dark theme
- Download the generated JSON with replaced placeholders
