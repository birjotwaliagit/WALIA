# WALIA

This repository contains a small web utility to generate Elementor-compatible JSON files from plain text case studies. Paste the raw text, and click **Generate JSON** to preview and download the filled template. A default template is bundled so no upload is required.

### Features

- Built-in Elementor template ready to use
- Paste raw case study text and parse it automatically
- Load an example case study for quick demo
- Toggle light or dark theme
- Download the generated JSON with replaced placeholders
- Preview the parsed data before download
